¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje de marcado estándar utilizado para crear páginas web. Es el componente fundamental de la World Wide Web y se utiliza para estructurar y presentar contenido en la web. HTML permite a los desarrolladores web organizar texto, imágenes, enlaces, multimedia y otros elementos en una página web de manera coherente y accesible.
